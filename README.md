# 4-Wheeler Service Centre Management System

A complete web-based management system designed for automobile service centres to handle customers, vehicles, bookings, billing, and payments efficiently. Built using **Node.js, Express, EJS, and MySQL**, it covers the full workflow from customer registration to service completion and invoice generation.
